easyhd

easyhd 旨在降低大数据中台后端开发简化对 Hadoop 的 API 操作，让开发从繁琐的代码抽离出来,只需要关注业务的实现，easyhd 是一款依赖JDK.Proxy以及Spring Boot轻量级无侵入的 Hadoop 客户端包含 HDFS 客户端,Yarn 客户端封装,对 Hadoop 组件的 API 封装, 操作 Hadoop 简单调用定义一个接口、一个方法就可以完成操作。
