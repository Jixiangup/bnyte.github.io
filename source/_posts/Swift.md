# 组件使用

## Button(按钮)

> 参考: https://developer.apple.com/documentation/swiftui/button

### 声明

```
struct Button<Label> where Label : View
```

### 概述

您可以通过提供操作和标签来创建按钮。该操作是一种方法或闭包属性，当用户单击或点击按钮时它会执行某些操作。标签是描述按钮操作的视图——例如，通过显示文本、图标或两者

```swift
Button(action: {}) {
    Text("Sign In")
}
```
