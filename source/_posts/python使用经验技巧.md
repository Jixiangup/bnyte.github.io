# Python 使用包管理

- 安装 PIP

```bash
# 下载安装脚本
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
# 运行安装脚本
sudo python3 get-pip.py
# 确认是否安装成功
pip --version
```
